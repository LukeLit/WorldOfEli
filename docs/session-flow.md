# Session Flow (You + Eli + Viktor)

Goal: get to a tiny playable game loop in one sitting.

## 0) Pre-Call (10 minutes)

- Open the `WorldOfEli` landing page.
- Open Slack channel with Viktor ready.
- Start screen capture before Eli joins.

## 1) Intro (5 minutes)

- Introduce Eli to Slack and Viktor.
- Ask Eli to tell Viktor what kind of game he wants.
- Mirror his ideas back in simple terms.

## 2) Pick The One-Session Scope (10 minutes)

Guide Eli to one tiny scope:

- One player action
- One goal
- One restart button

Example script:

- "What is the one thing your character does?"
- "How do we know you win?"
- "How do we start over fast?"

## 3) Build Loop (30-45 minutes)

- Create the minimum playable version.
- Keep momentum: build, test, celebrate small wins.
- Defer polish ideas to a "Next Session" list.

## 4) Publish + Demo (5 minutes)

- Push to GitHub.
- Confirm Vercel deploy.
- Have Eli play the first version and narrate what is fun.

## 5) Wrap + Journal (5 minutes)

- Add one short session entry in `journal/`.
- Capture:
  - what we built
  - what worked
  - what Eli wants next

## Facilitation Prompts

- "Pick one: jumping game or collecting game?"
- "Should we make it easier or harder?"
- "What do you want to name this world?"
